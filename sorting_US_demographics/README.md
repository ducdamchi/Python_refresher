************* SORTING ALGORITHMS ON US DEMOGRAPHICS DATA *************

Last modified: April 2023

Author: Duc Dam

About: This project implements BubbleSort and SelectionSort on data about US Demographics.
It allows user to interact with the program, narrowing down their search on a specific region of the US, where they can obtain information about its population.

***** Instructions *****

To run, type in shell: python3 sort_demographics.py

The program will pre-load a data on US demographics by State and County names. User can then choose the following options:

    1. Filter records by state name
    2. Filter records by total population
    3. Sort by state name
    4. Sort by total population
    5. Reset list to all records

The user can navigate freely among these options to find the desired information.
